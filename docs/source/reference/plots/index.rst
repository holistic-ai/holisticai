=============
Plots
=============

This page covers all plots available in `holisticai.plots` module.

The following subpackages are public.

- ``holisticai.plots.bias``: Functions and classes for bias metrics and results visualizations.
- ``holisticai.plots.explainability``: Functions and classes for explainability metrics and results visualizations.
- ``holisticai.plots.robustness``: Functions and classes for robustness metrics and results visualizations.
- ``holisticai.plots.security``: Functions and classes for security metrics and results visualizations.
- ``holisticai.plots.efficacy``: Functions and classes for efficacy metrics and results visualizations.
