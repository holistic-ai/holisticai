=============
Explainability
=============

Explainability in machine learning refers to the ability to understand and interpret the decisions and predictions made by AI models. It is crucial for ensuring transparency, trust, and accountability in AI systems.

This page provides an overview of all public ``holisticai`` objects, functions, and methods. All classes and functions exposed in the ``holisticai.*`` namespace are public.

Below is a list of modules in ``holisticai``:
