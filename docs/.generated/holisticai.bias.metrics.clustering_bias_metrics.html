holisticai.bias.metrics.clustering_bias_metrics(group_a, group_b, y_pred, data=None, centroids=None, metric_type='equal_outcome')

Clustering bias metrics batch computation.

This function computes all the relevant clustering bias metrics, and displays them as a pandas dataframe.
